# Resume_Checker_Agent

📄 Resume Checker AgentAn automated, AI-powered pipeline designed to evaluate resumes directly through Telegram. This project leverages automation platforms (like Make.com) to bridge Telegram, Google Docs, and AI agents, providing users with instant feedback and emailed evaluation reports.🚀 Key FeaturesTelegram Integration: Real-time interaction via a dedicated Telegram bot.  Automated Resume Parsing: Extracts content directly from Google Doc links using Regex and Google Docs API.  AI-Driven Evaluation: Utilizes a specialized Resume Evaluator Agent to analyze experience, skills, and formatting.  Multi-Channel Delivery: Sends evaluation summaries via Telegram and detailed reports via Gmail.  Context-Aware Bot: Handles everything from initial greetings to processing email addresses for report delivery.  🛠️ System ArchitectureThe project consists of four primary automated workflows:1. Telegram Watch Updates (The Brain)This is the entry point of the system. It listens for incoming Telegram messages and triggers the Resume Evaluator AI Agent.Triggers: Telegram Webhooks.  Logic:Identifies greetings and provides instructions.  Detects Google Doc links to initiate evaluation.  Captures email addresses to finalize report delivery.  2. Resume Content ExtractorHandles the technical heavy lifting of retrieving document data.Pattern Matching: Uses Regex to isolate the docID from a Google Drive URL.  Data Retrieval: Connects to the Google Docs API to pull the full text content of the resume for the AI to analyze.  3. Messaging EngineA dedicated module to ensure the bot can respond to users with specific information.Function: Sends plain text messages or evaluation snippets back to the user's Telegram Chat ID.  4. Email Delivery SystemThe final stage of the pipeline for professional report sharing.Integration: Connects to Gmail.  Action: Automatically drafts and sends an email titled "Resume Evaluation" containing the AI's full analysis.  📋 How It WorksUser Starts Chat: The user sends /start or a greeting to the Telegram bot.  Submit Link: The user provides a link to their resume (Google Doc).  Content Parsing: The system extracts the text and sends it to the AI Agent.  AI Analysis: The agent evaluates the resume based on hiring standards and generates a report.  Receive Results:A brief summary is sent via Telegram.  Upon providing an email, the full report is delivered to the user's Inbox.  ⚙️ Tech StackAutomation: Make.com  APIs: Telegram Bot API, Google Docs API, Gmail API  AI: Custom AI Agent (LLM-based)  Utilities: Regex (Regular Expressions)  🛠️ Setup InstructionsTelegram Bot: Create a bot via @BotFather and obtain your API token.Google Cloud: Enable Google Docs and Gmail APIs and set up OAuth2 credentials.Automation Setup: Import the provided JSON workflow files into your automation platform (e.g., Make.com).Webhook Configuration: Connect the Telegram Watch Updates module to your bot's webhook URL.  Agent Prompting: Ensure the Resume_Evaluater_agent is configured with the necessary system instructions for resume analysis.
